# AWS AI Services

This section covers Amazon Web Services' AI and Machine Learning offerings.

## Topics

- **Amazon-Bedrock**: Fully managed service for foundation models (FMs) via API
- **SageMaker**: Build, train, and deploy ML models at scale
- **Amazon-Q**: AI-powered assistant for business and development
- **Amazon-CodeWhisperer**: AI coding companion (now part of Amazon Q Developer)
- **Amazon-Rekognition**: Image and video analysis
- **Amazon-Comprehend**: Natural language processing service
- **Amazon-Lex**: Build conversational interfaces (chatbots)
- **Amazon-Polly**: Text-to-speech service
- **Amazon-Transcribe**: Speech-to-text service
