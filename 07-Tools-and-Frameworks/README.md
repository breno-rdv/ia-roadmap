# Tools and Frameworks

This section covers essential tools and frameworks for working with AI and LLMs.

## Topics

- **LangChain**: Framework for developing applications with LLMs
- **LlamaIndex**: Data framework for LLM applications (formerly GPT Index)
- **Hugging-Face**: Platform for sharing models, datasets, and ML tools
- **PyTorch**: Deep learning framework by Meta
- **TensorFlow**: Deep learning framework by Google

## Key Focus Areas

- Understanding when to use each framework
- Integration with AWS services
- Best practices for each tool
- Building production-ready applications
