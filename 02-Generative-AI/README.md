# Generative AI

This section covers various approaches to generative artificial intelligence and their applications.

## Topics

- **Concepts**: Core principles of generative AI, history, and key breakthroughs
- **GANs**: Generative Adversarial Networks - architecture, training, and applications
- **VAEs**: Variational Autoencoders - theory and practical uses
- **Diffusion-Models**: Stable Diffusion, DALL-E, and other diffusion-based models
- **Transformers**: Attention mechanism, transformer architecture (foundation for modern LLMs)

## Key Focus Areas

- Understanding different generative approaches
- When to use each type of model
- Current state-of-the-art techniques
